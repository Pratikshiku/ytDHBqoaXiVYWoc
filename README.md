### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (any of the listed software can be used)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Functional Description
The system is divided into (roles): regular users, administrators.

**Regular User Functions:**
1. Login
2. Registration
3. Homepage
4. Announcement Information
5. Messages
6. Personal Center
7. Change Password
8. Personal Information
9. Announcement Information Management
10. Payment
11. Messages
12. Maintenance Management
13. Travel Track Management
14. Carousel Information


**Administrator Functions:**
1. Login
2. Personal Center
3. Unit Number Type Management
4. Announcement Type Management
5. Maintenance Type Management
6. Visitor Management
7. Announcement Information Management
8. Payment Management
9. Message Management
10. Maintenance Management
11. Travel Track Management
12. Tenant Management
13. Carousel Information Management
